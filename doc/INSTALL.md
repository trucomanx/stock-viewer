# stock-viewer

Simple stock viewer program.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/stock_viewer/

```bash
pip install --upgrade stock_viewer
```

Using:

```bash
stock-viewer
```

## Install from source
Installing `stock-viewer` program

```bash
git clone https://github.com/trucomanx/StockViewer.git
cd StockViewer
pip install -r requirements.txt
cd src
python -m build
pip install dist/stock_viewer-*.tar.gz
```
Using:

```bash
stock-viewer
```

## Uninstall

```bash
pip uninstall stock_viewer
```
